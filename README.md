# A. K. Uustalu magistritöö

Siin on minu magistritöö failid. Failis "Uustalu_MA_andmestik.xlsx" on magistritöö andmestik koos märgendustega. Failis "Failid ja mida nad teevad.xlsx" on iga skripti nimi, sisend, väljund ja kirjeldus, mida see skript teeb. Kaustas "Programmid, sisendid ja tulemused on kõik skriptid", vajalikud sisendfailid ja väljundid.
